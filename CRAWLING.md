1. 결국 웹 사이트의 소스 코드를 분석
2. 스크래핑의 원리
   1. 웹 사이트 소스 코드 확인
   2. 추출하고 싶은 데이터의 위치 확인
   3. 해당 위치의 데이터를 추출하는 코드 작성

---

## AI에게 시킬 때

1. 웹 사이트의 소스 코드를 전달해야한다.
   1. 웹 브라우저의 요소 -> 전체 복사
   2. 브라우저 도구를 AI에게 사용하게끔 유도
      1. Playwright MCP
      2. Chrome DevTools MCP
      3. **Cursor Browser**
   3. 해당 브라우저 도구를 사용해서 내가 추출하고 싶은 사이트를 AI가 맥락으로 제공받을 수 있게 해야한다.
   4. 내가 원하는 요소를 AI에게 맥락으로 전달하여 `어떤 데이터를 어느 위치에 저장하고 싶다.`
2. 고급
   1. 브라우저가 HTML, JS를 실행한다.
   2. 코드로 웹 사이트에 요청을 하게 될 경우 -> 정적 데이터 -> HTML에 이미 있던 데이터만 가져올 수 있음
   3. 정적/동적 페이지
      1. 정적: HTML 데이터를 의미
      2. 동적: JS -> HTML 요소를 동적으로 만든 사이트
      3. 브라우저를 키지 않고 스크래핑을 한다! -> 정적 페이지
   4. 스크래핑에 좋은 전략 -> 코드가 브라우저를 알아서 켜서 JS까지 렌더링(약간의 텀을 줘야 할 수 있음) -> 렌더링이 다 되면 그 때 코드로 가져오고 싶은 데이터로 가져오기
